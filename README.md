# Important Commands:

- Cria o ambiente virtual
    - ```python -m venv venv``` 

- Ligar o ambiente virtual

    - No cmd.exe
        - ``` venv\Scripts\activate.bat``` 
    - No PowerShell
        - ``` venv\Scripts\Activate.ps1``` 


- Instalar depencências:
    - ``` pip install -r requirements.txt``` 

- Atualizar dependencias
    - ``` pip freeze > requirements.txt```
